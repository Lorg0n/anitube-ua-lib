# anitube-ua-lib

## Description
- Parser for data from the site anitube.in.ua (EN)
- Парсер даних з сайту anitube.in.ua (UA)

## Required Libraries
- beautifulsoup4
- requests-cache

## Examples
```python
from anitube import AniTube

api = AniTube()

for anime in api.get_animes(limit=5):
    print(anime.name)
```
